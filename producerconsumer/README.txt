Producer Consumer Problem instructions

From within \producerconsumer, run with .\run.bat [Producers] [Consumers] [Runtime] [Producer Sleep Time] [Consumer Sleep Time]
Or from within the windows explorer, click on and run the run.bat file in the producerconsumer folder.
For example, you can copy/paset .\run.bat 16 1 10 30 60 into the command line from within \producerconsumer
